الأسئلة الشائعة

كيف يمكنني الإبلاغ عن خطأ في تعريف؟

يمكنك إرسال بريد إلكتروني إلى bugs@qamoos.org مع تحديد: الكلمة المعنية، المعجم، والخطأ المحدد. سنقوم بمراجعته وتصحيحه في أقرب وقت.

هل يمكنني اقتراح إضافة معجم جديد؟

نعم! نرحب بمقترحاتكم. أرسل لنا اسم المعجم ومعلومات عنه على info@qamoos.org. سنقوم بدراسة إمكانية إضافته.

كيف يمكنني استخدام القاموس في مشروعي؟

للاستخدام الشخصي والتعليمي، يمكنك استخدام القاموس مجاناً. للاستخدام التجاري أو دمج API في مشروعك، يرجى التواصل معنا على partnerships@qamoos.org.

هل توفرون API للمطورين؟

نعم، نوفر API عام محدود. للحصول على وصول موسع أو استخدام تجاري، يرجى التواصل معنا للحصول على مفتاح API خاص.

كيف يمكنني المساهمة في تحسين القاموس؟

يمكنك المساهمة من خلال:
• الإبلاغ عن الأخطاء
• اقتراح تحسينات
• مشاركة القاموس مع الآخرين
• المساهمة في الترجمة (إن توفرت)
